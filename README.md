### Who I am

Full-stack developer and web designer based in Florida. I work across the entire stack — backend logic, APIs, frontend design, deployment. I don't pick a side.

Currently building **[Vyloft](https://vyloft.com)** — EdTech software for schools that combines real-time student screen monitoring with a built-in learning platform. Made for teachers who actually need to know what's happening in their classroom.
